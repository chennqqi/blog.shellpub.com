漏洞预警-WordPress Core远程代码执行0day(无需验证和插件)

2017年5月1日，WordPress 被曝出远程代码执行（RCE）漏洞，国外legalhackers发布该漏洞利用视频，从被曝出的视频中可以看到该漏洞不需要验证，同时不需要安装任何插件即可获取反弹的shell，目前该漏洞官方尚未回应。另外，漏洞细节legalhackers并未公布。预计等待官方发布补丁后会放出。

WordPress是一款由PHP语言开发的内容管理系统（CMS）系统。是当今最流行的cms，很适合二次开发，使用量超过百万。潜在影响范围广泛，请大家在官方补丁出来之前，提早做好防范。以免带来不必要的麻烦。

修复建议

  1. 即时关注wordpress中文官方，官方发布补丁，第一时间修复该漏洞

  2. 推荐使用河马查杀软件http://www.shellpub.com等对网站webshell进行查杀,及时清理网站后门。

  3. 安装相关防护软件，如主机waf，或者云防护(网站卫士、加速乐)；如使用阿里云主机，可开启阿里云防护。
